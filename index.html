Politica de Privacidades do Site

Ridemap

A sua privacidade é importante para nós. Nesta Política de Privacidade, explicamos de forma clara como o Ridemap coleta, utiliza, armazena e protege os dados pessoais de seus visitantes.

Este documento, bem como todo o conteúdo do site, é oferecido por Henrique Dev, neste termo representado apenas por “RESPONSÁVEL”, e regula o tratamento dos dados pessoais dos usuários do site, aqui denominados “VISITANTES”, em conformidade com a legislação vigente, especialmente a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/18 – LGPD).

O repositório oficial do projeto está disponível em: https://github.com/DeathHapyness/RideMap.

A navegação e permanência no site implicam na leitura, compreensão e aceitação integral desta Política de Privacidade.

Esta Política foi atualizada pela última vez em 25/01/2026.

1. DA COLETA DE DADOS PESSOAIS

O Ridemap pode coletar dados pessoais do VISITANTE de forma direta ou automática, sempre que necessário para o funcionamento do site ou melhoria da experiência do usuário.

Os dados coletados podem incluir, mas não se limitam a:

  • • Nome e e-mail, quando fornecidos voluntariamente pelo VISITANTE
  • • Informações enviadas por meio de formulários ou contatos
  • • Endereço IP, tipo de navegador, sistema operacional e páginas acessadas

O RESPONSÁVEL compromete-se a coletar apenas os dados estritamente necessários, respeitando os princípios da finalidade, necessidade e transparência.

2. DA FINALIDADE DO USO DOS DADOS

Os dados pessoais coletados são utilizados para:

  • • Permitir o funcionamento adequado do site
  • • Melhorar a experiência de navegação
  • • Responder dúvidas, mensagens ou solicitações do VISITANTE
  • • Análises estatísticas e melhorias técnicas do projeto

Os dados não são utilizados para fins comerciais, venda ou compartilhamento indevido.

3. DO USO DE COOKIES

O site Ridemap pode utilizar cookies para melhorar a experiência de navegação do VISITANTE.

Cookies são pequenos arquivos armazenados no navegador do usuário que permitem reconhecer preferências e otimizar funcionalidades.

O VISITANTE pode, a qualquer momento, configurar seu navegador para bloquear ou excluir cookies, ciente de que isso pode impactar algumas funcionalidades do site.

4. DO COMPARTILHAMENTO DE DADOS

O Ridemap não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando:

  • • Exigido por obrigação legal ou ordem judicial
  • • Necessário para cumprir determinações legais ou regulatórias
  • • Para proteção dos direitos, segurança e integridade do RESPONSÁVEL

5. DA SEGURANÇA DOS DADOS

O RESPONSÁVEL adota medidas técnicas e organizacionais adequadas para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

Entre as medidas adotadas estão:

  • • Uso de conexão segura (HTTPS/SSL)
  • • Controle de acesso aos dados
  • • Monitoramento e atualização contínua do ambiente

Apesar de todos os esforços, nenhum sistema é totalmente seguro, não sendo possível garantir segurança absoluta.

6. DOS DIREITOS DO VISITANTE

Nos termos da LGPD, o VISITANTE possui os seguintes direitos:

  • • Confirmar a existência de tratamento de dados
  • • Acessar seus dados pessoais
  • • Solicitar correção de dados incompletos ou desatualizados
  • • Solicitar a exclusão de dados, quando aplicável
  • • Revogar consentimentos concedidos

Para exercer seus direitos, o VISITANTE deverá entrar em contato com o RESPONSÁVEL por meio dos canais disponíveis no site.

7. DO ARMAZENAMENTO DOS DADOS

Os dados pessoais são armazenados apenas pelo tempo necessário para cumprir as finalidades descritas nesta Política, respeitando as obrigações legais e técnicas aplicáveis.

Após esse período, os dados poderão ser excluídos ou anonimizados.

8. DAS ALTERAÇÕES DESTA POLÍTICA

O RESPONSÁVEL reserva-se o direito de alterar esta Política de Privacidade a qualquer momento, sem aviso prévio.

Recomenda-se que o VISITANTE revise este documento periodicamente para estar ciente de eventuais atualizações.

9. DAS DISPOSIÇÕES FINAIS

Esta Política de Privacidade é regida pelas leis da República Federativa do Brasil.

Fica eleito o foro da comarca do domicílio do RESPONSÁVEL para dirimir quaisquer questões relacionadas a este documento.

Esta Política entra em vigor a partir de 26/01/2026.
